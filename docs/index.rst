chemsource Documentation
========================

Welcome to chemsource's documentation!

chemsource is a tool for classifying novel drugs and health-related chemicals by origin. 
It provides functionality to retrieve chemical information from various sources including 
PubMed and Wikipedia, and classify compounds using AI language models.
